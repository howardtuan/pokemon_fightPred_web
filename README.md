# 寶可夢對戰預測
### 專案啟用步驟
#### 一、建立及啟用虛擬環境
##### 建立:
<pre>
  $ python3 -m venv pokemon
</pre>
##### 啟用:
<pre>
  $ source pokemon/bin/activate
</pre>
#### 二、下載套件
<pre>
  pip3 install -r requirements.txt
</pre>
#### 三、啟動Server
##### 進入與app.py同層的資料夾
<pre>
  在命令提示字元輸入"flask run"或"python3 app.py"
</pre>
#### 四、進入網站
<pre>
  在網址列輸入"http://127.0.0.1:5000/index"
</pre>
